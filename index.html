Hardware

Aqui está a descrição de todos os componentes essenciais para o funcionamento do robô.

O Arduino é o cérebro do robô, ele que conecta todos os outros componentes, para que tudo fique funcional e programável, e sua programação pode ser customizada para qualquer uso, e pode ser usado não só para programar um robo, mas tambem para automações e outros projetos que envolve eletrônica e programação.

O Módulo Bluetooth é conectado direto no arduino, para que o arduino receba entradas do usuario sem o uso de um cabo, mas sim um aplicativo no seu celular, que conversa com o código dentro do arduino, para que o robô faça uma de suas funções essenciais, o movimento.

O Shield Motor é oque faz a comunicação entre os motores e o arduino, sem eles a programação do arduino não saberia como se comunicar com os motores.
Os Motores existem para exercer as funções motoras do robô, essas sendo, o movimento das armas e o movimento das rodas.

E por fim, o Sensor de Fim de Curso, é conectado ao arduino no intuito de comunicar ao codigo quando que algo encostou no robô assim ativando as armas.
